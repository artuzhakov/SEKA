# SEKA — Полная техническая документация

Автор проекта: **Artem Uzhakov**  
Игра: **SEKA** — онлайн-реализация традиционной карточной игры.

Данный пакет документации описывает:

- Игровой процесс (раунды, статусы, переходы)
- Систему ставок и банка
- Режимы `DARK` и `OPEN`
- Механику `REVEAL`
- Механику **Свары**
- Таймеры
- Конечные автоматы (FSM)
- Статусы игроков и игры
- Систему комбинаций карт
- API и примерные сценарии

## Структура документов

- `GAME_STATES.md` — состояния игры
- `PLAYER_STATES.md` — состояния игрока
- `STATE_MACHINE.md` — конечные автоматы
- `ROUND_LOGIC.md` — логика раундов
- `BETTING_RULES.md` — система ставок
- `DARK_OPEN.md` — режимы DARK / OPEN
- `REVEAL.md` — механика REVEAL
- `SVARA.md` — механика Свары
- `TIMERS.md` — таймеры
- `CARD_SYSTEM.md` — карты, комбинации, джокер
- `API_REFERENCE.md` — API бэкенда
- `EXAMPLES.md` — сценарии игр

Фронтенд — «тонкий клиент» (только UI и запросы).  
Вся бизнес-логика реализуется на бэкенде (Laravel).
