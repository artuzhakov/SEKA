# SEKA — Механика Свары (SVARA.md)

(Сокращённая версия — полная логика уже согласована в диалоге.)

Свара запускается:

- если после SHOWDOWN есть ≥2 игрока с одинаковым максимальным количеством очков;
- победители голосуют за Свару;
- если «ЗА» > «ПРОТИВ», запускаются этапы подключения других игроков.

Этапы:

1. Голосование победителей (ничего не платят)
2. Голосование проигравших игроков (вход за `bank/2`)
3. Голосование игроков в статусе WAITING (вход за `bank`)

Банк Свары:

- стартует с текущего `bank` игры;
- увеличивается за счёт доплат;
- комиссия не списывается до окончательного завершения всех Свар.

Игровой процесс Свары — полноценная новая игра SEKA с теми же правилами.
